Bienvenido a mi pagina Web

1.¿Que es PHP?

  • Es un lenguaje de programación del lado del servidor, usando principalmente para crear páginas web dinámicas.
  • Se integra facilmente con HTML, CSS y JavaScript.

  • Es gratuito, de codigo abierto y muy utilizado en sitios web, junto con bases de datos como MySQL.

2. Cracteristicas pricipales

  • Lenguaje interpretado (se ejecuta en el servidor).
  • Compatible con la mayoria de servidores web (apache, Nginx, IIS).
  • Multiplataforma (Windows, Linus, MaOS).
  • Se combina con bases de datos para CRUD (crear, leer, actualizar, eliminar).
  • Muy usado en CMS como WordPress, Joomla o Moodle.
